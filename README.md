# mcpulsor-host

## Переменные окружения

- `SPRING_AI_OLLAMA_CHAT_TEMPERATURE` – температура генерации Ollama-чата (по умолчанию: 0.1)
- `SPRING_AI_OLLAMA_CHAT_TOP_K` – ограничение top-k для Ollama-чата (по умолчанию: 10)
- `SPRING_AI_OLLAMA_CHAT_TOP_P` – ограничение top-p для Ollama-чата (по умолчанию: 0.95)
- `SPRING_AI_OLLAMA_CHAT_REPEAT_PENALTY` – штраф за повторы в Ollama-чате (по умолчанию: 1.0)
- `MCP_CLIENT_BASE_URI` – базовый URI MCP-клиента (по умолчанию: http://localhost:8091)
- `MCP_CLIENT_ENDPOINT` – endpoint MCP-клиента (по умолчанию: /mcpulsor)
